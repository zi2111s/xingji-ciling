# 考研英语背单词APP - 产品需求文档

## 1. 产品概述

**产品名称**：研词通（YanCiTong）

一款专注于考研英语词汇记忆与阅读能力提升的移动端应用。通过导入考研单词书单，自动生成贴合真题风格的阅读短文，配合标准化阅读练习，帮助考生高效备考。

- **核心价值**：将孤立的单词记忆融入语境阅读，让考生在真实阅读情境中掌握考研核心词汇
- **目标用户**：准备参加全国硕士研究生招生考试（英语一/英语二）的考生
- **使用场景**：碎片化时间学习、通勤途中、每日打卡练习

## 2. 功能模块

### 2.1 核心页面

| 页面名称 | 模块组成 | 功能描述 |
|---------|---------|---------|
| 首页 | 今日任务、进度概览、快速入口 | 展示今日学习计划、总体进度、快捷入口导航 |
| 单词书 | 书单列表、章节选择 | 管理考研单词书、选择学习章节 |
| 阅读训练 | 短文阅读、答题界面 | 阅读短文并完成配套练习 |
| 历史记录 | 做题记录、错题本 | 查看历史练习、复习错题 |
| 我的 | 个人中心、设置 | 用户信息、学习统计 |

### 2.2 功能详情

#### 2.2.1 单词书导入模块
- 支持从预设列表选择考研单词书（英语一/英语二分类）
- 预设词库包含：5500+考研核心词汇
- 按章节分组，每章约50-80个词汇
- 单词数据结构：单词、音标、词性、中文释义、真题例句

#### 2.2.2 阅读生成模块
- 基于导入的章节词汇，自动生成150-200词的阅读短文
- 短文风格高度仿真，贴合历年真题阅读材料特点
- 短文中的目标词汇高亮显示
- 生成算法确保短文既包含目标词汇又通顺可读

#### 2.2.3 答题系统模块
每篇短文配套4道单选题，完全对标考研英语阅读题型：

| 题型 | 出题逻辑 | 选项设置 |
|-----|---------|---------|
| 细节题 | 考查对文中具体信息的理解 | 4个干扰项（混淆词、时态错配、无关信息） |
| 词义题 | 考查根据上下文推断词义的能力 | 4个词义相近但程度不同的选项 |
| 主旨题 | 考查对文章整体主旨的把握 | 4个涵盖不同侧重点的选项 |
| 推理题 | 考查根据文意进行合理推断的能力 | 4个需要不同程度推理的选项 |

每题包含：
- 题目文本
- 4个选项（A/B/C/D）
- 标准答案
- 详细解析（指出原文依据、排除干扰项的理由）

#### 2.2.4 历史记录模块
- **练习记录**：保存所有做过的短文、题目、答案、时长
- **错题本**：自动收录错题，按题型、难度、错误原因分类
- **复习功能**：支持针对错题的专项复习
- **数据统计**：正确率趋势、薄弱题型分析

## 3. 用户流程

### 3.1 首次使用流程
```
启动APP → 词书选择 → 选择学习章节 → 开始学习
```

### 3.2 每日学习流程
```
进入首页 → 查看今日任务 → 开始阅读练习 → 阅读短文 → 完成4道题目 → 查看答案解析 → 记录保存
```

### 3.3 错题复习流程
```
进入历史 → 选择错题本 → 选择要复习的错题 → 重新作答 → 查看解析 → 掌握程度更新
```

## 4. 数据统计与进度

### 4.1 进度追踪指标
- 累计学习天数
- 已完成短文数量
- 题目正确率
- 各题型正确率对比
- 单词掌握度

### 4.2 视觉进度展示
- 首页环形进度图（今日任务完成度）
- 单词书页面（章节完成状态标记）
- 历史页面（正确率折线图）

## 5. 技术约束

- **平台**：移动端优先（手机浏览器、PWA）
- **离线支持**：已练习内容支持离线查看
- **数据存储**：本地存储 + 云端同步（可选）
- **性能要求**：短文生成时间 < 2秒

## 6. 验收标准

- [ ] 可以成功导入并选择考研单词书章节
- [ ] 每篇短文包含且仅包含4道题目（细节题、词义题、主旨题、推理题各1道）
- [ ] 每道题目有且仅有1个正确答案，并附带解析
- [ ] 历史记录完整保存所有练习数据
- [ ] 错题本自动收录做错的题目
- [ ] 支持历史记录的查看和错题复习
